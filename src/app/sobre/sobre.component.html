Sobre o Desafio O Desafio

Em resposta ao desafio proposto, resolvi implementar uma aplicação simples, simulando uma agenda telefônica.

A aplicação foi dividida assim:

  • Menu lateral, no qual são direcionados para as telas descritas a seguir.
  • Tela inicial, onde seriam listados os contatos com alguns detalhes, os dados são estáticos a partir de uma array de objetos;
  • Tela de cadastro do contato em si, solicitando algumas informações pessoais, mas que não é persistido em lugar algum, a tela possui um botão de salvar com um dialog de sucesso e botão de cancelar, ambos os botões retornam para a tela inicial;
  • Tela do sobre, onde estar descrito uma breve explicação do que foi feito.

Para tanto, utilizei as ferramentas abaixo:

  • Angular
  • Angular Material
  • Visual Studio Code
  • Chrome
  • Github
